== Description ==

This plugin enables your website visitors to recharge their mobile linee and power utilities. All major telecommunication networks in Nigeria are supported while for power utilities are ranged within AEDC, IKEDC, KEDCO, IBEDC, PHEDc, KAEDCO, JED also within Nigeria. 
You can recharge your line in one simple step. Flexible payment options are also available. Your line is instantly credited once your transactions are completed.



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `iRecharge_plugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go through the Settings and select 'iRecharge Vendor Set' to set your Vendor ID, this is displayed on your profile at irecharge.com.ng/profile
4. put [irecharge] on any page area you want the plugin to appear.

== Frequently Asked Questions ==

= What is my Vendor ID

Your Vendor ID is a unique value assigned to your Vendor account on www.irecharge.com.ng. Your ID is prominently displayed on your Dashboard
